QUESTION 2: Why must a SOC analyst NEVER use "Bridged" mode when detonating unknown malware?
ANSWER: Bridged mode acts like an open door. It allow the VM to connect directly to the home network and the internet. Using it can allow malware to scan the
network and spread across household device or communicate with a Command and Control server. Host only mode ensures the malware remains isolated to the host
machine.
